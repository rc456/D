Practical No. 03
Aim: Create and Manage Cloud Storage

Step 1: Create a Bucket
Go back to the AWS Management Console in your browser.
In the top Search bar, type S3 and click the first result.
Click the orange "Create bucket" button.
Bucket Name:
The name must be globally unique. No two people in the whole world can have the same bucket name.
Try this pattern: my-cloud-practical-firstname-2025
Rules: No uppercase letters, no spaces.
AWS Region: Ensure it is still Asia Pacific (Mumbai) ap-south-1.
Object Ownership: Leave as "ACLs disabled" (Recommended).
Block Public Access settings for this bucket:
Keep this CHECKED.
Because This prevents random people on the internet from seeing your private files. We will learn how to share files securely later.
Scroll to the bottom and click "Create bucket".

Step 2: Uploading an Object (Files)
Create a test file: On your Windows Desktop, create a simple text file named cloud-test.txt and type "Hello from AWS" inside it.
Go to your Bucket: Click on the name of the bucket you just created.
Click the orange "Upload" button.
Click "Add files" and select your File.
Expand "Properties", Leave it on Standard.
Click "Upload" at the bottom.
You should see a green banner: "Upload succeeded".

Phase 3: Organization (Folders)
S3 is technically a "flat" structure (it doesn't have real folders like Windows), but the console mimics them to help us organize.
Click "Create folder".
Name it logs.
Click "Create folder".

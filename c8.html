Practical No. 8
Aim: Host a simple web application in the cloud

Phase 1: Create the Website File
3. Paste this simple HTML code:
<html>
<body style="background-color:skyblue; text-align:center">
    <h1>Welcome to My Cloud Website!</h1>
    <p>Hosted on AWS S3 by a TYBSc CS Student.</p>
</body>
</html>
4. Save the file as index.html

Phase 2: Configure the Bucket
1. Go to the S3 Console.
Create a new bucket:
Block Public Access settings:
Uncheck "Block all public access".
Click Create bucket.

Phase 3: Upload and Enable Hosting
1. Click your new bucket name.
2. Click Upload -> Add files -> Select your index1.html -> Upload.
3. Once uploaded, click the Properties tab (top menu).
4. Scroll to the very bottom to "Static website hosting".
Click Edit.
Static website hosting: Select Enable.
Index document: Type index.html.
8. Click Save changes.

Phase 4: The Final Policy
Click the Permissions tab.
2. Scroll down to Bucket policy and click Edit
JSON
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "PublicReadGetObject",
            "Effect": "Allow",
            "Principal": "*",
            "Action": "s3:GetObject",
            "Resource": "arn:aws:s3:::YOUR-BUCKET-NAME/*"
        }
    ]
}
4. Click Save changes.

Phase 5: Visit Your Website
Go back to the Properties tab.
Scroll to the bottom "Static website hosting" section.
You will see a link labeled "Bucket website endpoint".
Click it.
